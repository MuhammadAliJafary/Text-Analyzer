**Recursive Text Analyzer (Python)**

This project is a beginner-friendly implementation of **recursion in Python**.
It demonstrates how common text-processing tasks can be solved **without using loops**, only recursion.

The project is part of a lab focused on:

* Understanding recursion
* Practicing base cases and recursive cases
* Debugging recursive functions
* Applying recursion to strings and lists

**Project Structure**

```
text-analyzer/
│── analyzer.py          # Core recursive functions
│── test_analyzer.py     # Unit tests for each function
│── main.py              # Demo program
```
